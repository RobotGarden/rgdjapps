## User table:
* Unique ID
* Name (use fuzzy search)
* Key card number
* Cobot ID
* Family ID
* Notes
* Emergency contact information

## Family Table
Links family members through ID pairs
* Unique ID
* User Unique ID

## Class table
* Unique ID
* Class name

## Training Table
Links users to classes they have taken
* User ID
* Class ID

## Equipment Table
* Unique ID
* Training Unique ID
* Equipment inventory / maintentance information
